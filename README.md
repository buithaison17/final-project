# Dự án Chào mừng ngày Nhà giáo Việt Nam

Dự án được xây dựng nhằm tạo ra không gian trực tuyến để học sinh, sinh viên hoặc người dùng có thể gửi lời chúc, lời tri ân đến các thầy cô nhân ngày **20/11 – Ngày Nhà Giáo Việt Nam**.

## Công nghệ sử dụng

### 1. **ReactJS**

Dùng làm nền tảng phát triển giao diện người dùng:

- Tổ chức component rõ ràng, tách biệt
- Tái sử dụng logic và UI
- Tối ưu hiệu năng và trải nghiệm

### 2. **Tailwind CSS**

Dùng để xây dựng giao diện nhanh và dễ tuỳ chỉnh:

- Utility-first
- Responsive
- Tốc độ phát triển cao
- Không cần viết CSS nhiều

### 3. **Một số thư viện khác**

Dự án có thể sử dụng thêm các thư viện hỗ trợ:

- **React Router** – Điều hướng giữa các trang
- **Supabase** – Lưu trữ lời chúc và thông tin người dùng
- **Mui Design / Ant Design** – Icon UI, phân trang
- **Redux Toolkit** – Quản lý state
- **Reat Toastify** - Thông báo khi gửi lời chúc
- **React Animate Onscroll** - Hiệu ứng khi cuộn trang
- **React Typed** - Hiển thị gõ chữ

Và một số thư viện khác

## Tính năng

- Gửi lời chúc tri ân trực tuyến
- Hiển thị danh sách lời chúc
- Tìm kiếm người dùng
- Xem chi tiết lời chúc kèm:

  - Tên người gửi
  - Lớp / thông tin tùy chỉnh
  - Thời gian gửi (ví dụ: “1 phút trước”)

- Giao diện thân thiện, trực quan
- Hoạt động tốt trên cả máy tính và thiết bị di động

## Cấu trúc dự án (tham khảo)

```
src/
 ├─ components/
 ├─ pages/
 ├─ store/
 ├─ utils/
 ├─ assets/
 ├─ App.jsx
 └─ main.jsx
```

## Cài đặt & chạy dự án

```bash
# Cài dependencies
npm install

# Chạy dự án
npm run dev

# Build production
npm run build
```

## Lời cảm ơn

Dự án được xây dựng với mong muốn góp phần tri ân và lan tỏa những giá trị tốt đẹp của nghề giáo – nghề trồng người.

Chúc các thầy cô luôn mạnh khỏe, hạnh phúc và tiếp tục cống hiến cho sự nghiệp giáo dục.
